## Lillian: Servidor Discord de Creadores

### Paso 1: Crear servidor
- Ir a discord.com → Crear servidor → "For a community"
- Nombre: "La Neta Creators" (o el que dirección indique)

### Paso 2: Crear estructura de canales
| Categoría | Canal | Propósito |
|-----------|-------|-----------|
| 📢 Info | #bienvenida | Mensaje de bienvenida automático + reglas |
| 📢 Info | #anuncios | Actualizaciones oficiales de La Neta |
| 📢 Info | #faq | Preguntas frecuentes fijadas |
| 💬 General | #general | Chat libre |
| 💬 General | #presentaciones | Cada creador se presenta |
| 💰 Monetización | #tips-monetizacion | Consejos de monetización |
| 💰 Monetización | #casos-de-exito | Historias de éxito |
| 🎬 Servicios | #soporte-streaming | Dudas sobre Gyre/streaming 24/7 |
| 🎬 Servicios | #soporte-doblaje | Dudas sobre doblaje MLA |
| 🎬 Servicios | #soporte-expansion | Expansión de plataforma |
| 🔧 Soporte | #soporte-tecnico | Problemas técnicos con Elevn |
| 🔧 Soporte | #feedback | Sugerencias de mejora |

### Paso 3: Crear roles
| Rol | Color | Permisos |
|-----|-------|----------|
| @Admin | Rojo | Todo |
| @Team La Neta | Azul | Moderar, fijar mensajes |
| @Creador Activo | Verde | Acceso a todos los canales |
| @Nuevo | Gris | Acceso limitado (#bienvenida, #general, #presentaciones) |

### Paso 4: Configurar bot de bienvenida
- Usar MEE6 (free) o Carl-bot (free)
- Mensaje automático cuando alguien se une:
  > "¡Bienvenido/a a La Neta Creators! 🎬 Aquí es donde los creadores conectan, comparten y crecen. Preséntate en #presentaciones y explora los canales. Si tienes dudas, el equipo de @Team La Neta está para ayudarte."
- Auto-asignar rol @Nuevo

### Paso 5: Diseño visual
- Icono del servidor: Logo de La Neta (pedir a Dayana si no tienes)
- Banner: Diseño simple con tagline
- Emojis custom: si el plan lo permite

## Eugenia: Comunidad WhatsApp

### Paso 1: Crear Comunidad WhatsApp
- Desde WhatsApp Business → Comunidades → Crear comunidad
- Nombre: "La Neta Creators Community"
- Descripción: Breve explicación del propósito

### Paso 2: Crear sub-grupos
| Grupo | Propósito |
|-------|-----------|
| Anuncios | Solo admins publican — noticias oficiales |
| General | Chat libre entre creadores |
| Tips & Monetización | Consejos y estrategias |
| Soporte Elevn | Problemas técnicos con la plataforma |

### Paso 3: Preparar mensajes de bienvenida
- Mensaje fijado en cada grupo explicando el propósito
- Reglas básicas de convivencia

### Paso 4: Documentar links de invitación
- Generar link de invitación para cada sub-grupo
- Compartir con Mery (Equipo 2) para que invite a los 50 fundadores
